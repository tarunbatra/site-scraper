# Site Scraper
App which scrapes sites. Duh!

## How to use
```
SITE=http://tarunbatra.com npm start
```

## Environment Variables
|  ENV                |  Default   |     Description             |
|:--------------------|:-----------|:----------------------------|
| SITE                |            | Site to scrape              |
| CONCURRENCY_LIMIT   | 5          | Maximum concurrent requests |
